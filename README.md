## Features

- Calendar View
- Reminders
- Subjects : Submissions and Lectures
- Attendance Tracker
- Events Explore Page
- Emails upon Event Registration
- e-Library

---

## Requirements

- Node.js and Node package manager (npm)

---

## Installation & Setup

```sh
$ git clone https://github.com/RyanDsilva/college-kampus-frontend.git

$ npm i

$ npm run dev
```
